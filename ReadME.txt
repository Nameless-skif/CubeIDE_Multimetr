7)Инициализация SPI2 для управления 74HC595(Vcc = 2-6V, Iout = 25mA(на ножку))
		 МК			74HC595

		(MOSI) PB15		 DC(Вход для последовательных данных)
		(MISO) PB14		 -
		(SCK)  PB13		 SH_CP(Вход для тактовых импульсов)
               	       PB12		 ST_CP(Синронизация ("защелкивание") выходов)
			GND		 OE(Вход для переключения состояния выходов из высокоомного в рабочее)
			Vcc		 MR(Сброс значений регистра. Сброс происходит при получение LOW)

	     
	     (HC595_ST_CP2) PB8		ST_CP второй мк 74HC595("1" - для переключения, "0" - выкл )
	     (HC595_ST_CP3) PB4		ST_CP третьей мк 74HC595("1" - для переключения, "0" - выкл )
	     (HC595_ST_CP4) PB3		ST_CP четвертой мк 74HC595("1" - для переключения, "0" - выкл )
             (HC595_ST_CP5) PB14	ST_CP пятой мк 74HC595("1" - для переключения, "0" - выкл )
	     (HC595_ST_CP6) PB9 	ST_CP шестой мк 74HC595("1" - для переключения, "0" - выкл )
			(по умолчанию все на GND)

	        (HC595_MR)PA8           MR сброс(общий, всех мк), подвести ко входам MR всех микросхем(подача "0" - сброс)
			(по умолчанию подкл к 3.3В)
